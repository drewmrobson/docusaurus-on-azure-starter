# Overview

Docusaurus is a fast, React-based static site generator using Markdown for posts. This starter gets it running on Azure Static Apps.

## Setup

```bash
npx create-docusaurus@latest docusaurus classic
cd docusaurus
npx docusaurus start
```
*Don't forget to check your docusaurus folder into git*

See [docusaurus.io](https://docusaurus.io/) for site configuration options.

## Running Local

```bash
cd docusaurus
npm install
npx docusaurus start
```
